# 拼豆图案生成器

中文 | [English](README.md)

一个基于 Web 的拼豆图案设计工具，可以将图片转换为拼豆图案，支持自定义绘制、颜色选择、图案导出和打印功能。

**[在线体验](https://perler-beads-convertor.vercel.app/)**

## 功能特点

### 🎨 图片转拼豆
- 上传图片自动转换为拼豆图案
- 智能匹配最接近的拼豆颜色
- 支持自定义画布尺寸（最大 500×500）

### ✏️ 绘图工具
- **画笔** - 绘制单个拼豆
- **刷子** - 批量绘制多个拼豆
- **橡皮擦** - 清除拼豆
- **取色器** - 从画布上吸取颜色

### 🎯 画布功能
- 网格显示，每5格有加粗分割线便于计数
- 支持缩放（0.5x - 20x）和拖拽移动
- 显示行列坐标编号
- 撤销/重做功能（最多20步历史）

### 🔲 板块拆分
- 支持按拼豆板尺寸拆分大图案
- 显示拆分线和板块编号
- 可导出为多个独立文件

### 📤 导出与打印
- 支持 PNG（透明背景）和 JPG 格式
- 多种缩放比例可选（1x - 10x）
- 导出图片包含坐标编号和颜色代码
- 打印预览功能

### 📊 颜色统计
- 实时显示使用的颜色种类和数量
- 点击颜色可直接选中使用

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist/` 目录下。

## 使用说明

### 1. 导入图片
- 在左侧面板点击「导入图片」按钮
- 选择要转换的图片
- 图片会自动转换为拼豆图案

### 2. 调整画布
- 在左侧面板设置画布宽度和高度
- 使用预设尺寸快速设置（20×20、29×29、50×50）
- 勾选「显示分割线」可查看板块拆分

### 3. 编辑图案
- 使用顶部工具栏选择绘图工具
- 在颜色栏选择要使用的颜色
- 在画布上绘制或擦除
- 使用 Ctrl+Z 撤销，Ctrl+Shift+Z 重做

### 4. 导出图案
- 点击顶部「Export」按钮导出图片
- 点击「Print」按钮打印图案

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Canvas API** - 画布渲染和绘图

## 颜色系统

项目内置了 36 种标准拼豆颜色（P01-P36），包括：
- 基础色：白色、黑色、红色、蓝色、绿色、黄色等
- 扩展色：各种深浅的粉色、紫色、橙色、棕色等
- 特殊色：透明、发光色等

颜色匹配算法使用 RGB 色彩空间的欧几里得距离来找到最接近的标准拼豆颜色。
